OscScoreP5

A Library by zeroisnan for the Processing programming environment.
Last update, 06/06/2020.

Record and replay OSC messages, useful for non real time sketch rendering.

Contributed Libraries are developed, documented, and maintained by members of the Processing community. Further directions are included with each Library. For feedback and support, please post to the Discourse. We strongly encourage all Libraries to be open source, but not all of them are.

Download

Download OscScoreP5 version 0.0.2 (002) in .zip format.

Installation

Unzip and put the extracted OscScoreP5 folder into the libraries folder of your Processing sketches. Reference and examples are included in the OscScoreP5 folder.

Keywords. OSC,XML,rendering

Reference. Have a look at the javadoc reference here. A copy of the reference is included in the .zip as well.

Source. The source code of OscScoreP5 is available at GitHub, and its repository can be browsed here.
